# Master of Mathematics

Master of Mathematics is a free educational website designed to help students learn mathematics in a simple and effective way. This website provides well-organized notes, important formulas, and downloadable PDF resources for Class 11 and Class 12 students.

## 📘 Features
- Easy-to-understand maths notes
- Important formulas for quick revision
- Chapter-wise content for Class 11 and Class 12
- Free downloadable PDF notes
- Simple and clean user interface

## 🎯 उद्देश्य (Purpose)
The main goal of this website is to make mathematics easy and accessible for all students, especially those preparing for school exams and competitive exams.
